Official Angular wrapper for [`dragula`][4].

# Demo

[![demo.png][1]][2]

Try out the [demo][2]!

# Install

You can get it on npm.

```shell
npm install angular-dragula --save
```

Or bower, too.

```shell
bower install angular-dragula --save
```

# Usage

Refer to the documentation for [`dragula`][4].
